2018 Final Year Project (ITE)

The objective of this project is to provide a platform for ITE College Central students and teachers to
explore and learn more about health, sports and fitness.
This application is created using Android Studio.

  • Main Page

    Main page for our application. It allows user to choose from Student Login and Staff Login

  • Login Page

    Login page for our application. It differentiate Student users and Staff users as they have different privilege

  • Home Page

    Home Page for our application. It allows user to students to choose various opitons while using the application Our application have a feature that helps students calculate their IPPT score

  • IPPT Calculation Page

    By typing the number of reps they did for Push up and Sit up and their timing for 2.4km run. It can calculate the score for both Girls and Boys

  • Three Stations information

    Three station page where students can find the stations they are weak at and find information to improve the various station that they are weak in. They can also find the scoring chart table for IPPT. This can help them to figure out how many reps to get in order to pass IPPT or getting Silver or Gold.

  • Events

    By going through the events, Students can find out the upcoming events that is happening.
